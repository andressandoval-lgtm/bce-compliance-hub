# bce-compliance-hub

Plataforma web para entidades financieras, fintechs y agregadores de pagos que deben cumplir con reportes del BCE.
Integra calendario y WhatsApp para recordatorios, permite cargar insumos y genera reportes en formato y estructura exigidos por la normativa.

Para conocer el alcance y roadmap del MVP, consulta el documento [MVP_GUIDE.md](docs/MVP_GUIDE.md).
